# BERT Fine-Tuning for Sentiment Analysis

## Project Description

This project focuses on fine-tuning a DistilBERT model for sentiment classification using the IMDb movie reviews dataset.

Two experiments were performed:

1. Freeze all BERT layers
2. Fine-tune last 2 layers

The performance of both experiments was evaluated using accuracy, precision, recall, F1 score, and confusion matrix.

## DataSet used:
"IMDB Moview Review Dataset"
